# 잔류 - 분기 설계 요약

## 분기 원칙
이 게임의 분기는 마지막 선택 하나보다, 플레이어가 역 안에서 어떤 태도를 보여 왔는지에 따라 갈라지는 구조가 적합하다.

## 핵심 분기 축
### 1. 규칙 준수
이상한 안내와 금기를 얼마나 경계하고 따랐는가.

### 2. 오염
물, 유인, 반복 구조, 잔류 구역에 얼마나 잠식되었는가.

### 3. 조사도
메모, 문서, 공간 구조를 통해 역의 진실에 얼마나 가까워졌는가.

### 4. 유인 반응
잘못된 출구, 호명, 구조처럼 보이는 것에 얼마나 반응했는가.

## 구역별 분기 역할
### 0구역
- 기준점
- 평범한 현실에서 시작했다는 감각 제공

### 1구역
- 이상함을 대하는 기본 태도 결정
- 경계할지, 확인하려 다가갈지 갈림

### 2구역
- 규칙이 실제 생존과 연결됨을 체감
- 위험 앞에서 침착한지, 끌려가는지 드러남

### 3구역
- 가장 중요한 중반 분기 구간
- 반복 구조, 표지판, 개인화된 개입 대응이 핵심
- 루프 / 침묵 / 탈출의 기반이 여기서 갈림

### 4구역
- 판정 구간
- 지금까지 누적된 태도와 마지막 선택이 결말로 이어짐

## 엔딩 방향
### 탈출
- 규칙을 비교적 잘 지킴
- 오염이 낮음
- 진실을 어느 정도 파악함
- 마지막에 쉬운 유도보다 현실적인 탈출 경로를 택함

### 루프
- 반복 구조를 완전히 이해하지 못함
- 잘못된 출구 유도에 흔들림
- 마지막에도 출구처럼 보이는 길을 성급히 믿음

### 편입
- 오염이 높음
- 금기를 자주 어김
- 물, 호명, 실루엣, 유도에 반복적으로 반응함

### 오인 구조
- 생존 감각은 어느 정도 있음
- 하지만 진실을 충분히 파악하지 못함
- 구조처럼 보이는 것을 받아들임

### 침묵
- 규칙을 매우 잘 지킴
- 유인에 거의 반응하지 않음
- 마지막까지 어떤 개입에도 자신을 내어주지 않음
